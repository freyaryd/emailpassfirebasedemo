Set this up w/your account:

  1. Put your own Firebase credentials in the head of this document. (Currently, lines 10-22)
  2. Go to the "Authentication" tab in your Firebase console (Firebase website) and go to "Sign-in method." Enable email/password
  3. Make an account, and check in the "Authentication" Firebase tab under "Users" that an account is created.

Using the demo:
  1. You NEED to create an account to log in. You NEED a "properly" formatted email, ie something@something.something, and a password.
  2. If there's something wrong w/email or password, there will be an error in the console. When you're using this LOOK AT THE CONSOLE it is your friend.
  3. Click the "display current user info" button to show the current user's info, ie who is logged in (will display email)

Firebase reference: https://firebase.google.com/docs/auth/web/password-auth
